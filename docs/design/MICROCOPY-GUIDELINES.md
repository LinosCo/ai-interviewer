# Microcopy Guidelines

## Tono

- Italiano chiaro, professionale, operativo.
- Evitare tecnicismi anticipati se l'utente non ha ancora contesto.
- Ogni testo deve aiutare l'utente a capire il prossimo passo.

## Pattern approvati

### Intro pagina

- prima frase: cosa controlli o decidi qui
- seconda frase: quale azione concreta puoi fare dopo

### Badge e stato

- il badge da solo non basta
- aggiungere sempre testo di supporto se lo stato implica un'azione

### Helper text

- breve
- specifico
- orientato all'effetto

Esempi:

- `Collega una destinazione per trasformare i tip in esecuzioni reali.`
- `Rivedi i tip canonici prima di automatizzare il routing.`
- `Questa sezione serve a capire cosa misurare, non solo a leggere numeri.`

## CTA

- usare verbi diretti
- massimo 3 parole quando possibile
- evitare duplicati come `Vai`, `Apri`, `Gestisci` se il contesto non cambia

Preferenze:

- `Apri connections` per setup integrazioni
- `Configura routing` per destinazioni e policy
- `Rivedi tip` per lavorare sul tip canonico
- `Misura impatto` per leggere analytics

## Regole per inglese vs italiano

- mantenere inglese se fa parte dell'IA del prodotto
- tradurre in italiano se il termine e puramente descrittivo

Tenere:

- `Overview`
- `Listen`
- `Tips`
- `Execute`
- `Measure`
- `Strategy`
- `Connections`
- `routing`
- `Copilot`

Tradurre:

- `ready` -> `pronto`
- `failed` -> `da correggere`
- `manual only` -> `solo manuale`
- `approval` -> `approvazione`

## Pattern di checklist

- titolo del passo
- spiegazione implicita nel contesto, non nella riga
- CTA specifica

Esempio:

- `Attiva il primo tool`
- `Collega una connessione`
- `Rivedi il primo tip`
- `Configura una regola di routing`

## Pattern di guidance popup

- titolo: un'azione
- descrizione: cosa fare qui
- `Perche conta`: una frase
- CTA unica

## Pattern analytics

- headline: cosa sta succedendo
- spiegazione: impatto strategico
- CTA: cosa fare adesso

## Accessibilita copy

- non usare solo colore per comunicare stato
- i pulsanti devono avere testo autosufficiente
- placeholder non sostituisce label o helper text
